# screenshots

Add product screenshots here for the README's Screenshots section.
Suggested filenames (referenced from `README.md`):

- `hero.png` — Hero section (light mode)
- `app-light.png` — Upload / compression panel (light mode)

compressed with Qompres itself before committing.
